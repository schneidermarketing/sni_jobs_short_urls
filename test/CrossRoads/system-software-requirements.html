Desktop Requirements

Browsers/Browser Settings

  • Our applications have been tested for optimal presentation on Internet Explorer 7.0 on Windows XP, and on Firefox 5.0+ and Google Chrome 12.0+.
  • There are currently issues with Internet Explorer 9.0 that we are addressing.
  • Javascript must be enabled
  • Cookies must be enabled
  • XMLHTTP support must be enabled.
  • ActiveX support must be enabled.
  • IE 7 to run with patch http://support.microsoft.com/kb/282402
  • Adobe Flash
  • Popup blockers should be disabled for Schneider sites

Resolution

  • Our applications have been optimized for 1024x768
